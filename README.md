Commands:
<br>
Run the app:
<code>node app.js</code>
<br>
To Add a note:
<code>node app.js add --title "name"</code>
<br>
To list all the notes:
<code>node app.js list</code>
<br>
To read note:
<code>node app.js read --title "name"</code>
<br>
To remove note:
<code>node app.js remove --title "name"</code>
